## Data Notice & Usage

**Dataset Source:** Brain Tumor MRI Dataset (Kaggle)

Please download the dataset from the original source (e.g., Kaggle) and arrange the folders as follows:

data/
├── Training/
│ ├── glioma
│ ├── meningioma
│ ├── pituitary
│ └── no_tumor
└── Testing/
├── glioma
├── meningioma
├── pituitary
└── no_tumor



⚠️ Only the folder structure and a sample note are included in this repository — **the actual image data is not uploaded** due to licensing and size constraints.
